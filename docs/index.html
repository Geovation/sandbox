Welcome to Geovation Sandbox

  • Stations Noise impact

    This web application aims to measure the noise impact to buildings near any random railway station in the UK and Ireland within a 500m radius. This project used the content of the OS Vector Tile API in the "Road" style using the Tile matrix Set EPSG:27700, which included the following features names.

    • The Open Zoomstack_Railway_Stations
    • The Open Zoomstack_District_Buildings

    You can test the application by visiting Stations Noise impact. You simply select a station from the dropdown menu and it generates the following information related to the station

    • The total number of buildings within the radius
    • Their total area
    • The noise impact obtained by the ratio of buildings area to that of the radius
